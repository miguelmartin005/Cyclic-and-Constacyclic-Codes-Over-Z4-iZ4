On our paper Cyclic and Constacyclic Codes Over Z4+iZ4 we present an algorithm to factor the polynomial x^n - lambda for all n odd and units lambda over the ring R = Z4+iZ4. This is then used to obtain constacyclic codes over R, which are then mapped to Z4 codes using a Gray map. Here we present the Magma script implementing both the factoring and the Gray maps.

The function ConstaLift(n,lambda) takes an odd length n and a unit lambda and returns the factorization of x^n-lambda over R[x].

Then the function ConstaCyclicCode(n,lambda,gen) for some divisor gen of x^n-lambda returns the lambda-constacyclic code generated by the polynomial gen.
CodeGrayMap(n, lambda, gen) returns the Z4 code corresponding to the image of this constacyclic code under the Gray map phi. If one wishes to use the alternative map phi', the instances of WordGrayMap inside this function can be changed to WordGrayMapAlt.

Finally, the function ExtensiveSearch(n) takes an odd n and computes all possible Z4 images of constacyclic codes of length n, and collects the best parameters found. This way the new best codes presented in the paper were obtained.
